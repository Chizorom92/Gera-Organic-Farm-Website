# GERA Organic Farm Website

## Overview

This is a multi-page static website for `GERA Organic Farm`, built using HTML, CSS, and a small amount of vanilla JavaScript. The project showcases a farm business with pages for home, about, services, contact, training, and a registration confirmation screen.

## Structure

- `project2 HTML/`
  - `index.html` — Homepage with hero section, farm overview, and navigation.
  - `ABOUT.html` — About page with farm story, organic farming methods, and benefits.
  - `SERVICE.html` — Services page describing farm offerings.
  - `CONTACT.html` — Contact page with form and contact details.
  - `TRAIN.HTML` — Training page for farm training information.
  - `confirm.html` — Registration confirmation page.
  - `README.md` — This documentation file.

- `project 2 CSS/`
  - `style.css` — Global styles for the homepage and shared layout.
  - `about.css` — Styles specific to the About page.
  - `service.css` — Styles specific to the Service page.
  - `contact.css` — Styles specific to the Contact page.
  - `train.css` — Styles specific to the Training page.
  - `confirm.css` — Styles for the registration confirmation page.

- `MEDIA/`
  - Image assets used across the site.

## Features

- Responsive mobile navigation with a sidebar menu and overlay.
- Shared header navigation across pages.
- Sectioned content for farm information, benefits, and organic methods.
- Button-based page navigation to support calls to action.
- Use of external Font Awesome icons.

## How to Use

1. Open any `.html` file in a browser.
2. Navigate the site using the top menu or mobile sidebar.
3. Use the training form links to reach the `TRAIN.HTML` page.

## Notes

- The project is entirely client-side and does not require a server.
- All styling is handled through the CSS files under `project 2 CSS/`.
- The mobile sidebar is controlled by simple JavaScript inside each HTML page.
